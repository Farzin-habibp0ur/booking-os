## How to Use This File

1. **Trend Scout** uses this as fallback when Apify/Firecrawl MCPs are unavailable
2. **Content Strategist** cross-references this for evergreen brief ideas during slow news weeks
3. Each trend has a **Pillar**, **Funnel Stage**, and **Shelf Life** rating
4. Trends marked `PERENNIAL` can be used any time; `SEASONAL` trends have date windows

## Usage Notes

- When using evergreen trends as fallback, flag output with `[CACHED]` or `[EVERGREEN]`
- Do NOT use more than 3 evergreen trends in a single briefing (risk of repetitive content)
- Cross-check against last 4 weeks of `briefings/` to avoid topic repetition
- Refresh keyword data when MCP sources come back online
