# Platform Gate Checker

> Used by the Performance Analyst agent weekly to evaluate whether Phase B or C platforms should be unlocked.
> Reference: [platform-launch-config.md](./platform-launch-config.md)

---

## Instructions

1. Run this check every Monday as part of the weekly performance review.
2. Fill in current metrics from each platform's native analytics.
3. All criteria in a phase must be met to recommend unlocking.
4. If ANY criterion is below threshold, recommend REMAIN and note the gap.
5. When a phase is unlocked, update `platform-launch-config.md` status from LOCKED to ACTIVE with the unlock date.
