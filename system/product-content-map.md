## Content Production Rules

1. **Every blog post** MUST reference at least 1 specific feature with screenshot/recording reference
2. **Every BOFU social post** MUST show actual product UI (not stock images)
3. **Video scripts** MUST specify which screen recording to use with timestamp ranges
4. **Comparison pages** MUST use real feature data, not vague claims
5. **When a feature is updated** in the product, flag existing content for refresh
6. **TOFU content** may reference features generally but doesn't require specific screenshots
7. **Cross-reference** `system/quality-gates.md` Gate 3 for mandatory product mention checks
